## API ENDPOINTS

### Płatności
| Endpoint | Metoda | Opis |
|----------|--------|------|
| `/api/payments/stripe` | POST | Tworzenie Stripe Checkout Session |
| `/api/payments/paynow` | POST | Inicjowanie płatności PayNow |
| `/api/payments/zen` | POST | Inicjowanie płatności Zen.com |

### Webhooks (dla bramek płatniczych)
| Endpoint | Metoda | Opis | Rejestracja |
|----------|--------|------|-------------|
| `/api/webhooks/stripe` | POST | Stripe events | dashboard.stripe.com |
| `/api/webhooks/paynow` | POST | PayNow IPN | panel.paynow.pl |
| `/api/webhooks/zen` | POST | Zen IPN | merchant.zen.com |

|----------|--------|------|
